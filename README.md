# ToDoTerm
Minimal ToDo list for terminal.

### Interactive CLI

```
$ ToDoTerm
```

## Dependencies
1. ```gcc```
2. ```make```

## Installation

### Build from Source
```
$ git clone https://github.com/diomedescerda/ToDoTerm.git
$ cd ToDoTerm
$ make
```
Then copy ```bin/ToDoTerm``` to your ```$PATH```.
